# WyJS_Base
A mininal skeleton for a javascript demo, game, application which need full screen canvas

## Support :
  - HTML5
  - Javascript
  - Double-buffering
  - Screen orientation
  - Events:
    - mousedown
    - mouseup
    - click
    - touchstart
    - touchmove
    - touchend
  - Minimal animation
    - drawAnimatedMessage(message)
  - Helpers
    - drawCenter()
    - drawCenteredLines()
    - drawCenteredDiagonals()
    - drawMessageTopLeft
    - drawMessageBottomRight
  - Logs in console can be manage by switching the booleans
    - debug -> basic debug
    - debugXY -> print the coordinates on events

## Requirements
  - jquery-x.x.x.js  -> https://www.libsdl.org
  - full-page-canvas.min.js -> Which allow you the full page canvas
